# MapTilesDownloader_Node

#### 介绍
MapTilesDownloader的node版本, 页面借用MapTilesDownloader项目的，因为原项目下载文件不能控制gizp所以自己就写了node版本下载作为替代
可以现在 高德地图瓦片 谷歌地图 Mapbox 等等， 支持自定义地址
 

#### 使用教程

1.  npm install
2.  node index.js
3.  打开 http://localhost:9100/index.html
